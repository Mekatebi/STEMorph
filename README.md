# STEMorph: Morphed Emotional Face Stimuli

[![DOI](https://img.shields.io/badge/DOI-10.1101%2F2024.05.13.593881-B31B1B.svg)](https://doi.org/10.1101/2024.05.13.593881)

**STEMorph** is a validated set of morphed emotional faces on a continuous **angry → neutral → happy** axis, derived from [NimStim](https://danlab.psychology.columbia.edu/content/nimstim-set-facial-expressions). It uses neutral-anchored morphing and neural-network face masks to avoid the artifacts of conventional morphing, giving researchers a controlled stimulus set for studying facial emotion recognition.

The set contains **198 faces** (22 identities × 9 morph steps). Subjective ratings track the intended morph level closely (mixed-effects β = 0.74, *p* < .001) with high test-retest reliability (ICC = 0.86). This repository holds the behavioral data, the MATLAB/Psychtoolbox experiment, and the Python analysis that reproduces the preprint's results.

## Accessing the stimuli

The face images are **not redistributed here** (NimStim terms). Request access via the [NimStim Set of Facial Expressions](https://danlab.psychology.columbia.edu/content/nimstim-set-facial-expressions); the morphed STEMorph set is then available from the corresponding author on reasonable request.
